1. 구글 폼 상단 **응답** 탭 클릭

## 🚀 3단계: Streamlit 배포 (Streamlit Cloud)

### 방법 1: GitHub 연동 (추천)

1. **GitHub 저장소 생성**
   ```bash
   # 로컬에서
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/survey-dashboard.git
   git push -u origin main
   ```

2. **Streamlit Cloud 접속**
   - https://share.streamlit.io 접속
   - GitHub 계정 연결
   - New app → 저장소 선택 → app.py 선택
   - Deploy 클릭

3. **배포 완료**
   - 약 2-3분 후 URL 생성: `https://your-app.streamlit.app`

### 방법 2: 로컬 실행

```bash
# 의존성 설치
pip install -r requirements.txt

# 실행
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 접속

## 📱 4단계: 특강 당일 활용

### QR 코드 생성
1. 구글 폼 URL 복사
2. QR 코드 생성기 (예: qr-code-generator.com) 사용
3. 슬라이드에 QR 삽입

### 실시간 대시보드 공유
1. Streamlit Cloud URL을 QR로 생성
2. 또는 대시보드를 프로젝터로 직접 표시
3. **자동 새로고침** 활성화 (사이드바에서 체크)

### 팁
- 특강 시작 전 사전 설문 응답을 먼저 받아두기
- 특강 중 추가 응답이 들어오면 30초마다 자동 업데이트
- 워드클라우드가 실시간으로 변화하는 모습을 보여주면 효과적

## ❓ 문제 해결

### 구글 시트 연결 안 됨
- 시트가 **"링크가 있는 모든 사용자"**에게 공개되어 있는지 확인
- URL 형식이 올바른지 확인

### 한글 워드클라우드가 깨짐
- 나눔 폰트 설치 필요 (로컬 실행 시)
  ```bash
  sudo apt-get install fonts-nanum
  ```
- Streamlit Cloud에서는 대체 시각화(막대 차트)로 표시됨

### 데이터가 업데이트 안 됨
- 브라우저 새로고침
- 사이드바에서 "자동 새로고침" 체크 확인
